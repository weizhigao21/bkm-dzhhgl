# 宝可梦多账号管家 `v1.2`

Pokemon66 免费加速服务的多账号管理工具，支持一键登录、多训练师切换、兑换码使用、订阅地址复制等功能。

## 快速开始

```bash
git clone https://github.com/weizhigao21/bkm-dzhhgl.git
pip install requests
python gui.py
```

或直接下载 [dist/宝可梦多账号管家.exe](dist/宝可梦多账号管家.exe) 运行。

## 功能

| 功能 | 说明 |
|------|------|
| 🔑 账号登录 | 邮箱 + 密码一键登录，自动获取 Token |
| 👥 多训练师管理 | 支持多账号切换，数据本地持久化 |
| 🗑️ 右键删除 | 训练师卡片右键即可删除，自动保存 |
| 📊 订阅查询 | 查看套餐名称、到期时间、流量使用等 |
| 📋 复制订阅地址 | 一键复制到剪贴板 |
| 🎫 兑换码使用 | 输入兑换码一键使用，无需手动操作 |
| 📜 运行日志 | 弹窗查看实时日志，支持清空 |
| 🚀 启动加载 | 打开程序自动显示第一个训练师信息 |
| 🖥️ 屏幕居中 | 启动时窗口自动居中显示 |

## 打包说明

```bash
pyinstaller --onefile --windowed --icon=1.ico --add-data "1.ico;." --name "宝可梦多账号管家" gui.py
```

输出文件在 `dist/` 目录下。将 `.exe` 放到任意文件夹运行即可，配置自动保存在同级目录。
